Идејата е во map методот да се излистаат сите ip adresi по месец посебно. 
Reduce ги собира сите ip адреси и ги става во set со што се бришат дупликатите. Исто така се ставаат во set и сите фајлови. Ако бројот на уникатни ip адреси е еднаков со бројот на уникатни фајлови, значи таа ip адреса направила request до секој фајл.
При ова би се добило output каде за секој месец се добиваат уникатните ip адреси кои правеле барања до сите фајлови од серверот.
Не можам да го прикачам целиот output бидејќи moodle не дозволува над 10mb, па прикачив првите неколку линии. Исто така поради некоја причина става null на почеток, но незнам зошто.
